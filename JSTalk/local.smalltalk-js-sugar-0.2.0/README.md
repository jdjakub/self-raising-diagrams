# Smalltalk Message Syntax for JavaScript
All of this VS Code extension (include this readme) was generated by Claude Opus 4.5.

Syntax highlighting and auto-conversion for Smalltalk-style message sends in JavaScript using Unicode mathematical brackets:

```javascript
⟦obj myMessage: arg1 with: arg2⟧   // keyword messages
⟦obj doSomething⟧                   // unary messages
⟦⟦obj get: key⟧ process: value⟧    // nested messages
```

## Auto-conversion

Type `[[` and it automatically converts to `⟦`. Type `]]` and it converts to `⟧`.

For nested JS arrays, use a space: `[ [1,2], [3,4] ]`

## Installation

1. Copy this folder to your VS Code extensions directory:
   - **Linux/macOS**: `~/.vscode/extensions/local.smalltalk-js-sugar-0.2.0`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\local.smalltalk-js-sugar-0.2.0`

2. Restart VS Code

## What gets highlighted

- **Message brackets** `⟦ ⟧` - marked as punctuation
- **Keyword selectors** like `myMessage:`, `with:` - highlighted as function names
- **Unary selectors** like `⟦obj doIt⟧` - highlighted as function names
- Everything else inside the brackets uses standard JS highlighting

## Customising colours

Add to your `settings.json`:

```json
"javascript.validate.enable": false,
"editor.semanticHighlighting.enabled": false,
"editor.tokenColorCustomizations": {
  "textMateRules": [
    {
      "scope": "entity.name.function.selector.smalltalk",
      "settings": { "foreground": "#DCDCAA" }
    },
    {
      "scope": "punctuation.definition.message-send.begin.smalltalk",
      "settings": { "foreground": "#FFD700" }
    },
    {
      "scope": "punctuation.definition.message-send.end.smalltalk",
      "settings": { "foreground": "#FFD700" }
    }
  ]
}
```
